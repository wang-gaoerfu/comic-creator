# AI漫画生成器 (Comic Creator)

使用 AI 生图技术创作漫画故事的工具集。支持剧情策划、分镜设计、多张图片生成（保持人物一致性），最终组装成可翻页阅读的 HTML 漫画站。

## 功能特性

- **剧情策划** - 根据主题自动创作漫画剧本
- **分镜设计** - 灵活的篇幅支持（短篇3-5张、中篇6-8张、长篇10+张）
- **人物一致性** - 定义主角外观，确保多张图中角色外观一致
- **单图生图** - 每张图都是完整画面，无宫格无分格
- **HTML阅读器** - 可翻页的漫画网页，支持键盘翻页
- **一键打包** - 右上角按钮打包下载全部图片

## 快速开始

### 方式一：在 aily 中使用（推荐）

1. 将 `comic-creator.skill` 文件导入 ailly
2. 对助手说"创作漫画"、"画个漫画"、"做个漫画"
3. 提供故事主题或完整剧本
4. 等待生成漫画并下载

### 方式二：独立使用 HTML 阅读器

```bash
# 1. 准备好漫画图片（panel01.jpg, panel02.jpg...）
# 2. 下载 assets/comic-reader/ 下的文件
# 3. 修改 index.html 中的图片路径
# 4. 用浏览器打开 index.html
```

## 使用示例

### 示例1：创作新漫画

**用户说：** "帮我做一个职场逆袭的漫画，10张左右"

**AI 创作剧本：**

| 张 | 场景 | 对话 |
|:--:|------|------|
| 1 | 办公室 | 领导："这季度数据就这？你是不是脑子有问题？" |
| 2 | 会议室 | 领导："重做！明天交！做不到就滚！" |
| ... | ... | ... |

**生成图片 → 组装 HTML → 打包下载**

### 示例2：提供故事直接生成

**用户说：** "帮我把这个故事画成漫画：
1.小王被领导当众羞辱
2.小王忍气吞声
3.小王发现领导贪污证据
4.小王举报领导
5.领导被开除"

**AI 直接生成5张图 → 组装 HTML → 打包下载**

### 示例3：短篇漫画

**用户说：** "画个搞笑反转的漫画，3张就够了"

**AI 创作剧本 → 生成3张图 → 组装 HTML → 打包下载**

## 输入输出

### 输入

| 字段 | 说明 | 示例 |
|------|------|------|
| 故事内容 | 用户提供的故事或主题 | "职场逆袭"、"创业励志" |
| 角色设定 | 主角外观描述（可选） | "25岁男性，戴眼镜，穿西装" |
| 篇幅要求 | 几张图 | "10张左右"或"3-5张短篇" |
| 风格要求 | 画面风格（可选） | "温暖治愈"、"搞笑"、"写实" |

### 输出

| 内容 | 说明 |
|------|------|
| 分镜剧本 | 表格形式（序号、场景、画面、对话） |
| 漫画图片 | 每张图独立文件 (panel01.jpg...) |
| HTML阅读器 | 可翻页的漫画网页 |
| ZIP包 | 打包下载 |

## 智能流程

1. **用户提供完整剧本** → 直接进入生图步骤
2. **用户只提供主题** → 先创作剧本，再生图
