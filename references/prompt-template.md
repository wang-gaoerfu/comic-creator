# AI漫画生图提示词模板

## 核心原则：单图！单图！单图！

**重要的事情说三遍：**
- ❌ 不要三宫格
- ❌ 不要四宫格
- ❌ 不要上下分格
- ✅ 每张图必须是完整的单个画面

---

## 人物一致性模板

每张图必须使用完全相同的人物描述：

```
25岁中国男性，圆脸，黑色短发，戴黑框眼镜，
穿深蓝色西装外套+白色T恤，表情[描述]
```

## 单图提示词公式

**每张图只用这个公式：**

```
一张[风格]漫画，画面是[完整场景描述]，
[人物1完整外观]+[具体动作]+[表情]，
[人物2完整外观]+[具体动作]+[表情]（如有），
[背景细节]，[色调]，[质感]
```

## 反转漫画技巧

### 第1-3张：铺垫（压抑/困境）
- **色调：** 冷蓝、灰暗
- **人物：** 主角处于弱势，被欺负
- **场景：** 办公室被骂、深夜加班、被不公平对待

### 第4-6张：转折（意外/希望）
- **色调：** 开始变暖，淡黄
- **人物：** 意外发生，转机出现
- **场景：** 做梦惊醒、得知好消息、出现转机

### 第7-10张：高潮+结局（反转/释然）
- **色调：** 明亮温暖，金橙色
- **人物：** 强烈反转，出乎意料
- **场景：** 反派倒台、真相大白、主角释然

## 输出格式

每张图生成后：
1. 保存为 `panel01.jpg`, `panel02.jpg`... 格式
2. 检查是否为单图（无宫格）
3. 按顺序组装到HTML阅读器
4. 打包成zip发给用户
